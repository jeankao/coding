GitHub

GitHub是一個利用Git進行版本控制、專門用於存放軟體代碼與內容的共享虛擬主機服務。截止到2015年，GitHub已經有超過九百萬註冊用戶。已經成為了世界上最大的代碼存放網站和開源社群。

https://github.com/

CodeAnywhere

CodeAnywhere是一個基于瀏覽器的程式编辑器，可以使用Github的帳號登入，目前支持 HTML、Python、JavaScript、CSS、和 XML 的語法高亮。

https://codeanywhere.com/

建立開發環境

(1)登入CodeAnywhere後，建立一個Python專案

(2)開啟終端機，下達安裝django的指令

sudo pip3 install django==2.0.2

(3)下載「我的數位助理」程式碼，在終端機中下達Git指令

git clone https://github.com/huechingkao/assistant.git

(4)試著啟動網站，在終端機中下指令



cd assistant
sudo python3 manage.py runserver 0.0.0.0:80

網站已經可以順利啟動了，接著我們要用瀏覽器來觀看這個網站。

網址在「Python」按右鍵，選取「Info」頁面下方就可以看到超連結。

(5)在我們的網站首頁，使用下列帳號進入系統

帳號：admin

密碼：admin2246

現在網站可以正常執行了，試試看它的功能吧!
